# Clean Architecture

## What is it?

This app follows the [Clean Architecture](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html) which is a variant of the [Hexagonal Architecture](https://en.wikipedia.org/wiki/Hexagonal_architecture_(software)).
